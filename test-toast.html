React Hot Toast Test

Open the app in the iframe below and test toasts:

Instructions:

  1. Open browser dev console and run: window.testToast = (type, msg) => { if(type==='success') toast.success(msg); else if(type==='error') toast.error(msg); else toast(msg); }
  2. Then call: window.testToast('success', 'This is a success toast!')
  3. Or: window.testToast('error', 'This is an error toast!')

Expected Result:

  • Toasts should appear in the bottom-right corner (not full screen!)
  • They should be compact and unobtrusive
  • Dark mode: toasts should have dark background (#2a2a2a) with light text
  • Light mode: toasts should have light background (#ffffff) with dark text
  • Success toasts show green icon, error toasts show red icon
  • Toasts auto-dismiss after 4 seconds
